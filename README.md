# Exploring-and-Visualizing-a-simple-dataset

This repository demonstrates basic dataset exploration and visualization with **pandas**, **matplotlib**, and **seaborn**.

## What it does

The script:
- Loads a CSV dataset using pandas
- Displays dataset structure using:
  - `df.shape`
  - `df.columns`
  - `df.head()`
- Creates basic visualizations:
  - Scatter plot
  - Histogram
  - Box plot

## Files

- `visualize_dataset.py` — main script
- `data/simple_dataset.csv` — sample dataset
- `plots/` — output directory for generated plot images

## Run

```bash
python visualize_dataset.py
```

After running, the generated plots will be saved in the `plots/` folder.
